## 1. ランチャー・生産性向上

### Raycast（無料/有料）⭐ 最強ランチャー

**公式サイト**: https://www.raycast.com

Mac のスポットライト（Spotlight）を大幅に強化したランチャーアプリです。アプリ起動、ファイル検索、計算、スニペット管理など多機能を1つのショートカットで呼び出せます。

**主な機能**:
- アプリ・ファイルの高速起動
- クリップボード履歴管理
- スニペット展開（テキスト補完）
- Window Management（ウィンドウ操作）
- 豊富な拡張機能（GitHub、Notion、Google など連携）
- AI アシスタント機能（Pro プラン）

**使い方**:
```
⌘ + Space（デフォルト）でランチャー起動
```

| 操作 | ショートカット |
|------|----------------|
| ランチャー起動 | `⌘ Space` |
| クリップボード履歴 | `⌘ Shift V` |
| スニペット | `!` をタイプ後に検索 |
| ファイル検索 | `find` と入力 |

:::message
Alfred の代替として非常に人気です。基本機能は無料で十分使えます。
:::

## 2. ウィンドウ管理

### Rectangle（無料）⭐ まず入れるべき定番

**公式サイト**: https://rectangleapp.com

ウィンドウをキーボードショートカットで素早く配置・リサイズするアプリです。Magnet の無料代替として非常に人気。

**主な操作**:

| 操作 | ショートカット |
|------|----------------|
| 左半分 | `⌃ ⌥ ←` |
| 右半分 | `⌃ ⌥ →` |
| 最大化 | `⌃ ⌥ ↩` |
| 左上1/4 | `⌃ ⌥ U` |
| 右上1/4 | `⌃ ⌥ I` |
| 中央配置 | `⌃ ⌥ C` |

**使い方の例**:
- ブラウザを左半分、エディタを右半分に配置して2画面作業
- Zoom 画面共有中に資料を右、Zoom を左に整理

---

### Magnet（有料 ¥480）

**App Store**: Mac App Store で購入可能

Rectangle の有料版に相当。シンプルで安定した動作が魅力。ドラッグでのスナップにも対応。

---

### Moom（有料）

ウィンドウのカスタム配置を保存・再現できる高機能ツール。複数モニター環境でも便利。

## 3. ターミナル・開発環境

### iTerm2（無料）⭐ 開発者必須

**公式サイト**: https://iterm2.com

macOS 標準ターミナルの強力な代替。分割ペイン、豊富なカスタマイズ、tmux 統合などが使えます。

**主な機能**:
- 画面分割（縦・横）
- タブ管理
- 検索ハイライト
- トリガー（出力に応じて自動アクション）
- Python API による自動化
- tmux 統合

**設定のおすすめ**:
```
Preferences > Profiles > Colors > Color Presets
→ Solarized Dark や Dracula がおすすめ
```

**よく使うショートカット**:

| 操作 | ショートカット |
|------|----------------|
| 垂直分割 | `⌘ D` |
| 水平分割 | `⌘ Shift D` |
| タブ切替 | `⌘ 数字` |
| ペイン移動 | `⌘ [ / ]` |

---

### Homebrew（無料）⭐ 必須パッケージマネージャ

**公式サイト**: https://brew.sh

macOS のパッケージマネージャ。開発ツールやアプリを CLI から一元管理できます。

**インストール**:
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

**よく使うコマンド**:
```bash
# アプリのインストール（CLI ツール）
brew install git node python

# GUI アプリのインストール（Cask）
brew install --cask rectangle iterm2 raycast

# インストール済み一覧
brew list

# アップデート
brew update && brew upgrade

# 不要なファイルを削除
brew cleanup
```

---

### Warp（無料/有料）

AI 搭載のモダンターミナル。コマンド提案、エラー説明、自然言語でのコマンド生成が可能。

## 7. ファイル管理

**公式サイト**: https://yazi-rs.github.io

Rust 製の高速ターミナルファイルマネージャ。vim ライクな操作でファイルを管理でき、プレビュー機能が優秀。

**インストール**:
```bash

```

**基本操作**:

| 操作 | キー |
|------|------|
| 移動 | `h j k l` |
| ファイル開く | `Enter` |
| 前のディレクトリ | `h` |
| 検索 | `/` |
| コピー | `y` |
| 貼り付け | `p` |

---

### Path Finder（有料）

Finder の高機能代替。デュアルパネル表示、カスタムツールバー、アーカイブ操作など。

---

### AppCleaner（無料）⭐ アンインストール必須

**公式サイト**: https://freemacsoft.net/appcleaner/

アプリを完全に削除するツール。アプリ本体だけでなく関連設定ファイル・キャッシュも一括削除します。

**使い方**:
1. AppCleaner を起動
2. 削除したいアプリをドラッグ＆ドロップ
3. 関連ファイル一覧を確認して「削除」

## 9. メニューバー整理・ユーティリティ

**公式サイト**: https://www.macbartender.com

**使い方**:

CPU、メモリ、ネットワーク、ストレージの使用状況をメニューバーに表示するシステムモニター。

## Homebrew でまとめてインストール

```bash

brew install --cask \
  raycast \
  rectangle \
  iterm2 \
  visual-studio-code \
  obsidian \
  maccy \
  appcleaner \
  bitwarden \
  jordanbaird-ice \
  stats \
  arc \
  figma \
  zoom \
  slack \
  discord
```

3. **AppCleaner** - アプリを完全削除できる

5. **1Password / Bitwarden** - セキュリティの基本

快適な Mac ライフをお楽しみください！
